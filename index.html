How to use this calculator

Fill out each input in order by row starting with "Starting Reading" then "Ending Reading" and so on.

In order to add the standing charge you must first add the dates for your readings, these will be used to calculate the number of days between the them and will add the standing charge to each day.

Reading a dial meter

If you have an old dial meter remember you must read the dials by alternativing between clockwise and anti-clockwise.
